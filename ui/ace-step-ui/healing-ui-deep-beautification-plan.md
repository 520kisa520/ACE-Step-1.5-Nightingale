# 疗愈风 React UI 深度美化计划

基于 ace-step-ui React+Vite+Tailwind 架构，专注纯音乐疗愈生成与 LoRA 数据集标注、微调的前端深度美化，不改动 ACE 模型接口通信逻辑。

## 阶段 1：环境配置与组件库集成

### 1.1 安装依赖
- 安装 ShadCN UI 组件库（基于 Radix UI + Tailwind）
- 安装 Chakra UI 核心组件
- 安装 Magic UI 动效组件
- 安装 Storybook 开发环境
- 配置 Tailwind CSS 主题扩展

### 1.2 配置开发环境
- 初始化 Storybook 配置
- 设置组件开发工作流
- 配置主题切换系统

## 阶段 2：主题系统重构

### 2.1 低饱和疗愈主题
- **夜色主题**：深绿色系 (#5C8A7A, #3D6A5A) + 柔和深色背景
- **晨曦主题**：浅绿色系 (#7A9E8E, #5C7A6E) + 柔和浅色背景
- 配置磨砂玻璃效果 (backdrop-blur)
- 自定义阴影系统 (healing, healing-strong)

### 2.2 动效系统

## 技术栈确认

- **框架**: React 19.2.4 + Vite 6.2.0
- **样式**: Tailwind CSS + 自定义主题
- **组件库**: ShadCN UI + Chakra UI + Magic UI
- **开发工具**: Storybook
- **图标**: Lucide React
- **类型**: TypeScript 5.8.2

## 关键注意事项

1. **不改动 ACE 模型接口**：保持现有 API 通信逻辑不变
2. **保持疗愈模式兼容**：确保 HEALING_MODE 相关功能正常
3. **渐进式重构**：分阶段实施，每个阶段可独立测试
4. **组件复用优先**：优先使用开源组件库，减少自定义开发
5. **性能优先**：动效和视觉效果不能影响性能
